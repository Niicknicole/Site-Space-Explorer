## Sobre

Space Explorer é um website interativo de múltiplas páginas com foco em temas de astronomia.  
O projeto foi desenvolvido com o objetivo de praticar estruturação front-end, design responsivo e interatividade com JavaScript.

O site contém seções como Missões, Sistema Solar, Buracos Negros e Galáxias.

---

## Destaques Técnicos

- Manipulação do DOM com JavaScript
- Lógica de randomização com controle de repetição
- Uso de Event Listeners
- Implementação de scrollIntoView()
- Layout responsivo utilizando Flexbox
- Estrutura modular com separação de responsabilidades
